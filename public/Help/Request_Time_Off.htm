Request Time Off 

Clicking a blank cell on a future date allows you to request time off, education, bereavement leave, jury duty, and other shift types that must be approved by your manager. 

To request time off:

  1. Click the Request tab.
  2. Select the time-off shift code from the drop-down list. For reference, your estimated PTO accrual displays in the request window.
  3. Select the start time.
  4. Select the duration of the request. The duration can be up to 12 hours per day, but should not exceed your normal shift length.  
  5. Enter a brief description or reason (optional).
  6. Click Add to Schedule. 
  • Pro Tip: When requesting consecutive days off, drag your cursor across the numbers in the calendar view. You can then enter your request once for all selected days. You can also deselect dates from within the selected range as needed to refine your request.
